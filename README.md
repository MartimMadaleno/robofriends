# Robofriends

A simple portfolio

## Installation

```bash
git clone https://github.com/MartimMadaleno/portfolio.git
```
OR
```bash
git clone git@github.com:MartimMadaleno/portfolio.git
```
AND
```bash
npm install
```

## Usage

To start the app

```bash
npm run start
```

To build the app

```bash
npm run build
```
To run tests on the app

```bash
npm run test
```
